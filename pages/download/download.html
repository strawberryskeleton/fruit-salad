Author's Note:

The project took a lot of research + trial and error to make. Making a color-glyph svg font was especially difficult and tricky and I had to put in a lot of work to finally figure out how to make a font in COLR-v2 format for free. I downloaded and tried multiple softwares, trying to debug and create the color svg font which took a lot of time.
All the effort I put into this project cannot be measured by the time I spent on just coding this website or making the custom font glyphs and assets. I learnt a lot from this project and for this project and I hope to coninue building more projects like this.
